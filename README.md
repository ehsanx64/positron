# positron

Positron (a.k.a antielectron) is an experimental web application backend written in Go.
  
## Features
- Delivery
    - net/http
    - gRPC
- Repository
    - sqlx/sqlite
- Messaging
    - MQTT
    - Websocket
- Caching
    - Redis\valkey
